# Worksheet Front-end

This is the front-end for the test task of a Worksheet.

Built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS**.

## Deployment

The app is available at https://test-task.771707.xyz

## Local development

```bash
npm install
cp .env.example .env
npm run dev
```

By default the app runs at `http://localhost:5173`.

> Set `VITE_API_BASE_URL` in `.env` (for example: `http://localhost:4000/api`).
